# MYSQL约束

| 分类       | SQL              | 备注                           |
| ---------- | ---------------- | ------------------------------ |
| 主键约束   | `primary key`    | 限定一个列，唯一且不为空       |
| 自增长约束 | `auto_increment` | auto_increment通常配合主键使用 |
| 非空约束   | `not null`       | 列的值不能为空                 |
| 唯一约束   | `unique`         | 列的值唯一                     |
| 默认约束   | `default`        | 默认值                         |
| 零值填充   | `zerofill`       | 零值填充                       |
| 外键约束   | `foreign key`    | 添加外键                       |
