# 社会主义发展及其规律


## 发展

英法两国的空想社会主义，代表人物`圣西门`，`傅里叶`，`欧文`，代表作`乌托邦`。

## 科学社会主义问世的标志

马克思，恩格斯，在1848年发表的《共产党宣言》，提出无产阶级革命。

## 无产阶级革命

- 无产阶级革命是人类历史上最广泛、最彻底、最深刻的革命。
- 建立了世界上第一个社会主义国家，苏维埃俄国
- 建立了马克思主义政党

## 马克思主义政党最终目标/根本宗旨

- 实现共产主义
- 为人民群众谋利益

## 形成了很多的社会主义

社会主义发展道路因每个国家国情不同，具有多样性，比如中国、古巴

社会主义是在实践探索中开拓前进的。
