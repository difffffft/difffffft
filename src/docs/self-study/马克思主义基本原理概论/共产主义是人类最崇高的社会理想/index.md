# 共产主义是人类最崇高的社会理想

考点：

```
q: 实现共产主义的条件

a：
    1.以社会主义为基础，社会主义是走向共产主义的必由之路
    2.社会生产力高度发展，是实现共产主义的根本条件和基础
```

```
q: 共产主义是啥样的

a：
    1.共产主义是按需分配的社会，而我们目前的社会主义是一个按劳分配的社会
    2.在共产主义社会，阶级通知和压迫会消灭
    3.在共产主义社会，大家自学树立起劳动的观念，并且乐于集体奉献，树立以集体主义为核心的共产主义的人生观，价值观，道德观。
```

```
q：共产主义远大理想和中国特色社会主义共同理想的关系

a:
    1.两者之间相辅相成，相互促进，有机的联系和统一
```
